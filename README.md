# MagnetHost - Trial Chambers Edition

Modern Minecraft hosting website frontend built for MagnetHost.

## Features

- Trial Chambers inspired UI
- Responsive modern design
- Minecraft themed landing page
- Pterodactyl dashboard integration
- Hosting pricing section
- Fast deployment support

## Tech Stack

- React
- Tailwind CSS
- Vite

## Deployment

This project can be deployed on:

- Cloudflare Pages
- Vercel
- Netlify
